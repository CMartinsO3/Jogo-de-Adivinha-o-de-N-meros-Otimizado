# Jogo-de-Adivinha-o-de-N-meros-Otimizado
Jogo de Adivinhação (1-100) em C focado em análise de dados. Oferece feedback alto/baixo, registra logs em arquivo e utiliza recursão para gerar estatísticas (média, melhor/pior sessão e desvio padrão). O objetivo é educacional: reduzir tentativas via sugestões de estratégia e ilustrar paradigmas de programação via terminal e multiplataforma.
